An implementation of a bao file, meaning some data blob with associated outboard.

Compared to just a pair of (data, outboard), this implementation also works when both the data and the outboard is incomplete, and not even the size is fully known.

There is a full in memory implementation, and an implementation that uses the file system for the data, outboard, and sizes file. There is also a combined implementation that starts in memory and switches to file when the memory limit is reached.
