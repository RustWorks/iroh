The NAT-PMP/PCP/UPnP prober/client, for requesting port mappings from NAT devices.

§no_v4_send: bool

Whether IPv4 UDP is known to be unable to transmit at all. This could happen if the socket is in an invalid state (as can happen on darwin after a network link status change).

§net_checker: Client

The prober that discovers local network conditions, including the closest relay relay and NAT mappings.

fn pick_relay_fallback(&self) -> Option<RelayUrl>

Returns a deterministic relay node to connect to. This is only used if netcheck couldn’t find the nearest one, for instance, if UDP is blocked and thus STUN latency checks aren’t working.

If no the RelayMap is empty, returns 0.

source

fn reset_endpoint_states(&mut self)

Resets the preferred address for all nodes. This is called when connectivity changes enough that we no longer trust the old routes.

source

async fn close_stale_relay_connections(&self)

Tells the relay actor to close stale relay connections.

The relay connections who’s local endpoints no longer exist after a network change will error out soon enough. Closing them eagerly speeds this up however and allows re-establishing a relay connection faster.
