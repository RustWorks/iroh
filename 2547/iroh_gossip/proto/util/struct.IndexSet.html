Struct iroh_gossip::proto::util::IndexSet

source · 
pub(crate) struct IndexSet<T> {
    inner: IndexSet<T>,
}
Expand description

A hash set where the iteration order of the values is independent of their hash values.

This is wrapper around [indexmap::IndexSet] which couple of utility methods to randomly select elements from the set.
